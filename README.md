# Cloud-Based AI Recruitment Platform on AWS

## Overview
This project demonstrates the design and deployment of a cloud-native AI recruitment platform on AWS. The platform was built for scalability, high availability, secure networking, and managed database operations.

---

## Technologies Used
- AWS EC2
- Amazon RDS (MySQL)
- Amazon VPC
- Application Load Balancer (ALB)
- Auto Scaling
- CloudWatch
- IAM
- Security Groups
- Linux
- Docker

---

## Key Features
- Designed a highly available AWS architecture across multiple Availability Zones
- Configured secure VPC networking with public and private subnets
- Implemented an Application Load Balancer to distribute application traffic
- Deployed services on EC2 / ECS Fargate
- Integrated Amazon RDS MySQL for managed database services
- Enabled monitoring and alerts using Amazon CloudWatch
- Secured infrastructure using IAM roles and Security Groups
- Supported AI-driven recruitment workflows:
  - Candidate scoring
  - AI interview simulations
  - Automated recruitment workflows

---

## Architecture
The platform uses a multi-tier AWS architecture with:
- Load balancing
- Managed database services
- Secure private networking
- Monitoring and alerting
- Scalable compute resources
